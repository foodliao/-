选择文件　新一个按钮搞定：点「选择文件」一次多选所有图片（同名前缀的多视角图会自动归为同一个物体），选完自动列出各物体组。

实时进度 + 自动重试　每组实时显示：等待 / 上传 / 生成中 / 下载 / 导入 / 已完成 / 失败。新网络抖动会自动重试，不会因一次抖动就判失败。

检查更新 · 版本号　新面板底部显示当前版本号，并有「检查更新」按钮。作者发布新版后会自动更新（见下文）。

自动更新（无需反复重装）
▲ 发现新版本 v2.1.0，正在更新…　/　已更新到 v2.1.0，重启 Blender 即可生效

自动检查

每次打开 Blender 加载插件时，后台静默检查有没有新版（不打扰操作）。检测到新版会自动下载并提示，重启 Blender 后新功能即生效。

手动检查

面板底部带当前版本号与「检查更新」按钮，想随时确认有没有新版，点一下即可。已是最新会提示「已是最新版」。

作者更新 LiClick 的生成能力后，你无需重新下载安装包——插件自己就会拉到最新版。

注意事项

贴图自带 · 无需外置

生成的模型以 .glb 内嵌贴图导入，颜色 / PBR 贴图随模型打包，存于 Blender 内部，不需要额外贴图文件夹。若需外置，可在「文件 → 外部数据 → 解包文件」导出。

需要 Node.js

上传与生成走 LiClick 命令行工具，依赖 Node.js v22+。安装器会自动检测，缺了会引导安装；装好后全自动，无需再管。

统一大小 · 自动排队

导入后每个模型缩放到统一大小、落到地面、沿 X 轴一字排开（按「排列间距」推进，单位 mm/cm/m 自动换算）。重名物体自动加 01/02 区分，不冲突。

图片格式 · 重试建议

支持 jpg / png / webp / bmp / tif；过小自动放大、过大自动缩小、超 4MB 转 JPEG。网络抖动会自动重试，建议单批不超过 20 组，失败项单独重试效率更高。
